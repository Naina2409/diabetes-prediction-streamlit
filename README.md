# Diabetes Prediction System

A Machine Learning web application that predicts whether a patient is diabetic using Support Vector Machine (SVM).

## Features

- Data Preprocessing
- Data Visualization
- Feature Scaling
- SVM Classification
- Streamlit Web Interface
- Real-Time Prediction

## Technologies Used

- Python
- Pandas
- NumPy
- Scikit-Learn
- Streamlit
- Matplotlib
- Seaborn

## How to Run

Install dependencies:

```bash
pip install -r requirements.txt
```

Run:

```bash
streamlit run app.py
```
